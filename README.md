# Deneme1-havadurumu

HTML, CSS ve JavaScript kullanarak geliştirdiğim, API entegrasyonu ile anlık veri çekme mantığını uyguladığım hava durumu projesi.

 🌤️ Hava Durumu Uygulaması (Weather App)

Bu proje, yazılım geliştirme yolculuğumda **JavaScript** ve **API** yapılarını anlamak için geliştirdiğim ilk çalışmalardan biridir. 

 🚀 Öne Çıkan Özellikler
**API Entegrasyonu:** Gerçek zamanlı hava durumu verilerini çekmek için [API] kullanıldı.
**Responsive Tasarım:** HTML ve CSS ile farklı ekran boyutlarına uyumlu arayüz.

 🛠️ Kullanılan Teknolojiler
**HTML & CSS:** Yapı ve stil yönetimi.
**JavaScript:** DOM manipülasyonu ve asenkron veri işlemleri.
* **REST API:** JSON formatında veri çekme ve işleme.

 💡 Neler Öğrendim?
Bu projeyi geliştirirken API dokümantasyonu okumayı, yapısını kullanmayı ve kullanıcıdan alınan veriye göre arayüzü dinamik olarak değiştirmeyi deneyimledim.
